Desafio de Array

Crie uma função que receba um array de números e retorne o maior número do array.

Exemplo:

                    Entrada: [3, 5, 7, 2, 8]
                    Saída: 8

Resolução

Para resolver esse desafio, você pode usar o método Math.max() em conjunto com o operador de espalhamento (...).

                        function encontrarMaiorNumero(array) {
                            return Math.max(...array);
                        }

                        // Testando a função
                        const numeros = [3, 5, 7, 2, 8];
                        console.log(encontrarMaiorNumero(numeros)); // Saída: 8

Maior sequência consecutivas de um array

Crie uma função que receba um array de números e retorne o comprimento da maior sequência consecutiva.

Exemplo:

                    Entrada: [1, 2, 3, 5, 6, 7]
                    Saída: 3 (sequência: 5, 6, 7)

Resolução

Para resolver esse desafio, você pode usar um loop para percorrer o array e contar a sequência consecutiva.

                        function maiorSequenciaConsecutiva(array) {
                            let maxCount = 0;
                            let currentCount = 1;

                            for (let i = 1; i < array.length; i++) {
                                if (array[i] === array[i - 1] + 1) {
                                    currentCount++;
                                } else {
                                    maxCount = Math.max(maxCount, currentCount);
                                    currentCount = 1;
                                }
                            }
                            return Math.max(maxCount, currentCount);
                        }

                        // Testando a função
                        const numeros = [1, 2, 3, 5, 6, 7];
                        console.log(maiorSequenciaConsecutiva(numeros)); // Saída: 3
